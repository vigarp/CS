Reconnaisance


Basic URL structure
1. protocol
    - https
    - http
    - ftp

2. host
    - subdomain (www)
    - rootdomain (example.com)

3. path
    - directory (/marketing/seo/)
    - file (index.php)

4. query string/parameter
    ?location=London&type=blog
    - ? => query string begin
    - location => parameter name
    - & => query string separator
    - blog => property value

5. fragment/hash => #chaptrer1
